# Markdown语法

标记 语法
标题1 # Heading 1
标题2 ## Heading 2
标题3 ### Heading 3
标题4 #### Heading 4
斜体 *italic*
粗体 **bold**
删除线 ~~line~~
图片 ![Image Title](http://xxx.png)
链接 [Link Text](http://xxx.com)
引用 > Here is a quote block
内联代码 `code`
块级代码 ```code block``` 或 ```javascript code block```
块级代码高亮行 ```js{3} code block```
导入代码片段 可高亮 <<< @/docs/yideng/week0/ES5.md{11}
分隔符 ---- 或 *****
生成目录 [[toc]]
锚点 [公式标题锚点](#标题)
表情 ✅ ❎ 💯 emojis列表

# 语义标签

语义标签 语法

斜体 <i>斜体</i>
加粗 <b>加粗</b>
强调 <em>强调</em>
上标 Z<sup>a</sup>
下标 Z<sub>a</sub>
键盘文本 <kbd>Ctrl</kbd>
