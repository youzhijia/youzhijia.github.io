# js手写代码实现1~10

# （1）new

实现原理

  1. 创建一个新对象
  2. 这个新对象会被执行 [[原型]] 连接
  3. 属性和方法被加入到 this 引用的对象中，并执行了构造函数中的方法
  4. 如果函数没有返回其他对象，那么 this 指向这个新对象，否则 this 指向构造函数中返回的对象
function new(func){
  let obj = {};
  obj.__proto__ = func.protptype;
  let res = func.call(obj);
  if(res && typeof(res) === 'function' || typeof(res) === 'object'){ 
    return res;
  }
  return obj;
}
