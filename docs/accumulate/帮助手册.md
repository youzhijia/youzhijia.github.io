# 帮助手册

## 部署发布项目

每次git pull后都有很多冲突，解决后部署不生效，浪费很多时间

#### 方法

* 在dev分支开发
* build && deploy
* 将dist更新的index.html内容复制粘贴到根目录index.html
* commit && push
* build && deploy
* commit && push
* 浏览效果
* dev分支继续开发
